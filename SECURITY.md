# Security Policy

## Reporting a Vulnerability

We take the security of our software seriously. If you believe you've found a security vulnerability, please report it by emailing dlucky171@gmail.com.

Do not disclose the issue publicly until we've had a chance to address it.

## Supported Versions

We currently support the latest major version of each project with security updates.

## Security Measures

- All commits are signed
- Dependencies are regularly audited
- Secrets are never committed to repositories
- Code reviews are required for all merges
